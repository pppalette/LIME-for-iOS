# LIME-for-iOS: Ad-killer for LINE iOS

## 概要

This is an Jailbreak Tweak to clean [**LINE**](https://line.me).

LINE を掃除する 脱獄Tweak です。

Inspired by [**LIME**](https://github.com/Chipppppppppp/LIME)

Make LINE Great Again

## 機能

* 各種広告の削除
* VOOM、通話、ニュースタブ削除
* ホームを昔に戻す(マイナーリージョン化)
